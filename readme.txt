== Description ==

All in One Facebook Pack allow you, as blog's admin, to manage facebook social plugins and open graph

Solved Some trouble

You can manage Social Plugins via Shortcodes or via php function.

Some Features of All in One Facebook Pack

*   Open Graph manager (you can disable it)
*   All in One Facebook Pack, apply the "Post Thumbnails" theme's support if your blog doesn't support it
*   You can create Facebook Ghost Pages
*   All Facebook Plugins supported
